В C# для округления значения до двух знаков после запятой можно использовать следующие методы:

1) Метод Math.Round: Этот метод округлит число до указанного количества знаков после запятой. 
Вам нужно будет передать исходное значение, а также количество знаков после запятой в качестве аргументов метода.

var roundedValue = Math.Round(value, 2);

2) Форматирование строки: Вы можете использовать форматирование строк для округления и отображения значения 
с определенным количеством знаков после запятой.

string.Format("{0:N2}", value);

3) Применение форматирования в строке формата: Для этого вам нужно создать строку формата с символом “N”, 
за которым следует количество знаков после запятой (в данном случае 2). Затем вы можете использовать эту строку 
формата для форматирования значения.

Console.WriteLine("The value is: {0}", value, "The value rounded is: {0:N2}", value);
